
# Angular-Node 

Este Proyecto contiene: 
- frontend en Angular
- Backend en Node



## Pasos para la Instalacion
  #### Descargar el repositorio
  
  ````
 git clone https://github.com/evancillo/docker-fiftech.git
  ````

#### Ubiquese en la Raiz del proyecto y Ejecute el docker compose 
````
docker-compose up
````
